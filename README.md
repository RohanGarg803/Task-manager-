# TaskZen ✦ AI Cat Companion & Task Manager

A futuristic, glassmorphic single-page task manager featuring **Mochi**, an interactive animated AI cat companion, and a persistent daily task completion streak widget.

## ✨ Features
- **Mochi the AI Cat**: Reacts to mouse clicks with jumps, meowing sounds, hearts, and expression winks. Falls asleep when idle.
- **Persistent Streak**: Flame widget (`🔥`) that tracks and pulses based on your consecutive daily completions.
- **Spotlight Task Input**: Raycast-inspired clean input console with glowing priority chips.
- **3D Card Hover Tilts**: Task cards tilt dynamically in 3D space based on mouse position.

## 🛠️ Tech Stack
- Built 100% with pure vanilla **HTML5, CSS3, and JavaScript**.
- Serverless, runs offline, and stores data in browser `localStorage`.
- Can be hosted for free on **GitHub Pages**.
